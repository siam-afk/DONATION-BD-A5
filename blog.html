Important FAQ

What is the Document Object Model (DOM)?

The DOM is a programming interface representing a webpage as a tree of objects, enabling dynamic updates.

How do you select an element from the DOM?

Use methods like getElementById("id"), querySelector("selector"), getElementsByClassName("class"), or getElementsByTagName("tag").

What is event delegation, and why is it useful?

Event delegation uses a parent element to manage events for child elements, improving performance and handling dynamic elements.

How do you manipulate an element's attributes and styles?

Use setAttribute() and getAttribute() for attributes, and element.style.property for styles (e.g., element.style.color = "red").
